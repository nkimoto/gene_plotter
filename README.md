
## Requirements
- R (>= 3.4)
- Install karyoploteR


## Install
To install karyoploteR, start R and enter:
```
## try http:// if https:// URLs are not supported
source("https://bioconductor.org/biocLite.R")
biocLite("karyoploteR")
```


## Usage
```
Rscript gene_plotter.R chrZ reference.txt orthlist.txt
```
